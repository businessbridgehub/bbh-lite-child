# BBH-Lite Child Theme

Welcome to the official child theme for **BBH-Lite**. 

A lightweight, performance-optimized, and clean child theme built for the **BBH-Lite** WordPress parent theme. This child theme provides a safe, clean, and structured environment for making code customizations, template overrides, and custom styling to the parent theme without losing your changes when the parent theme updates.

## About the Parent Theme
**BBH-Lite** is a minimalist, lightweight, and speed-optimized WordPress theme designed for developers, bloggers, and businesses who value clean code and performance. 
* **Original Theme:** [BBH-Lite on WordPress.org](https://wordpress.org/themes/bbh-lite/)

## 🚀 Features

* **Safe Overrides:** Built strictly following WordPress theme development standards.
* **Efficient Enqueuing:** Configured to dynamically load parent styles and establish proper dependencies without duplicate HTTP requests.
* **Performance Focused:** Inherits the core accessibility, speed, and responsiveness of the BBH-Lite theme ecosystem.
* **Ready for Customization:** Organized structure for adding custom hooks, functions, and CSS overrides.

## Why Use This Child Theme?
- **Keep Upgrades Safe:** Modify styles (`style.css`) and backend logic (`functions.php`) without worrying about future parent theme updates overwriting your work.
- **Performance First:** Inherits the lightweight, high-speed architecture of BBH-Lite.
- **Developer-Friendly:** Ready-to-use boilerplate structured specifically for the BBH-Lite ecosystem.

## 🛠️ Installation

1. **Prerequisites:** 
   Ensure you have the parent theme [BBH-Lite](https://wordpress.org/themes/bbh-lite/) installed in your WordPress `wp-content/themes/` directory.

2. **Download the Child Theme:**
   Clone this repository directly into your themes folder:
```bash
   cd /path-to-your-wordpress/wp-content/themes/
   git clone https://github.com/businessbridgehub/bbh-lite-child.git

```

**Or, Download from here:**
*Alternatively, download the [bbh-lite-child.zip](https://github.com/businessbridgehub/bbh-lite-child/releases/download/v1.0.4/bbh-lite-child.zip) architecture from GitHub and upload it via the WordPress Dashboard (**Appearance > Themes > Add New > Upload**).*

3. **Activation:**
* Navigate to your WordPress Admin Dashboard.
* Go to **Appearance > Themes**.
* Locate **BBH-Lite Child** and click **Activate**.



## 📂 File Structure

```bash
bbh-lite-child/
├── style.css       # Custom CSS overrides (enqueues parent styles automatically)
├── functions.php   # Custom PHP code, hooks, and filters
└── README.md       # Repository documentation
```
* `functions.php` – Handles asset enqueuing, style dependency configurations, and structural PHP customizations.
* `style.css` – Houses the child theme headers, metadata configuration, and custom CSS layout overrides.
